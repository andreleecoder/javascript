Template string é um novo método de concatenação entre strings e variaveis o a sintaxe é a seguinte `olar ${nome} turubom?`. Além de ficar estéticamente melhor, também permite a quebra de linha, coisa que com o antigo método de concatenação nao era possivel. Outra vantagem, é que fica mais fácil para realizar expressões matemáticas. Snitaxe:console.log(`7+7 é = ${7+7}`). É possivel executar funções com essa metodologia sintaxe: function calcular (x,y){ return x * y console.log(`A multiplicçao de 8 * 8 é: ${calcular(8,8)}`) }. Template string, abre uma janela dentro da string para a realização do javascript.
